정렬 알고리즘 — 단계별 동작

2과목

탭 눌러가며 각 정렬의 동작을 단계별로 확인해봐! 주황색 = 비교 중, 초록색 = 정렬 완료, 분홍색 = 피벗.

📌시험 포인트 ① 최악에도 O(n log n) 보장 → 힙, 병합 (퀵은 최악 O(n²))
② 추가 메모리 O(1) + O(n log n) → 힙 정렬만 해당
③ 안정 정렬 → 버블, 삽입, 병합, 기수

매뉴얼 작성 — 설치 vs 사용자

2과목
용어설명
설치 매뉴얼
설치 과정을 화면·메시지 순서대로 캡처해 작성. 설치 시작부터 완료까지 순차적으로.

컴포넌트별로 작성. 작성 절차: 작성 지침 정의 → 구성 요소 정의 → 구성 요소별 내용 작성 → 사용성 검토·검증.

비용 산정 — 비관치 · 낙관치 · 기대치 (LOC)

5과목
용어설명
낙관치
가장 적게 나올 것으로 예측한 라인 수 (제일 잘 풀렸을 때).
기대치
가장 많이 나올 확률이 높은(중간) 라인 수.
비관치
가장 많게 나올 것으로 예측한 라인 수 (제일 안 풀렸을 때).
예측치
공식 ★계산
 ( 낙관치 + 4×기대치 + 비관치 ) ÷ 6
예: (2,000 + 4×5,000 + 8,000) ÷ 6 = 5,000 라인
📌같이 나오는 공식노력(인월) = LOC ÷ 1인당 월평균 생산 라인 · 개발 기간 = 노력 ÷ 투입 인원 · PERT 일정 공식도 (낙관+4×기대+비관)/6 동일

포인터 — 최소 생존 3개념

4과목
개념설명
*p / &a
int *p = &a; → p에는 a의 주소, *p는 a의 값.
배열명
배열명 = 첫 요소의 주소 → a[i]와 *(a+i)는 완전히 같은 표현.
문자열
문자열 = char 배열 → 포인터로 한 글자씩 순회, 끝은 NULL(\0).
📌전략이중 포인터(**)·복잡한 포인터 연산 추적·함수 포인터 문제는 시험에서 패스 — 위 3개는 쉬운 배열·문자열 문제 판별용
